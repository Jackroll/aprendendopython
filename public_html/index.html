Comandos Git:

####### NAVEGAÇÃO #############

  1. pwd - Identifica o diretório que esta
  2. ls - lista os arquivos do diretório
  3. cd /d - Acessa o diretório D:
  4. cd pasta - Acessa pasta
  5. clear - limpa a tela

####### ARQUIVOS #############

  1. mkdir nomepasta - cria uma pasta
  2. touch nomearquivo - cria arquivo
  3. touch pasta/arquivo - cria pasta e arquivo
  4. git status - verifica se tem algum arquivo para fazer add ou commit
  5. git add nomedoarquivo - faz o add ao containner
  6. git add * - adiciona todos os arquivos disponiveis para o containner
  7. git commit -m "Comentário" - faz o commit dos arquivos disponiveis
  8. git log - lista todas as alterações sofridas nos arquivos
  9. git log --oneline - lista todas as alterações sofridas nos arquivos em apenas uma linha por modificação
  10. git log --graph - mostra o desenho do grafo de commit
  11. git log --oneline --graph - mostra o desenho do grafo de commit, somente os nodos
  12. git commit --am "comentários" - adiciona ao container e faz o commit de um arquivo ao mesmo tempo
  13. git branch - identifica o ramo do arquivo
  14. git diff - mostra as ultimas alterações no arquivo
  15. git checkout nomedoarquivo - descarta a ultima alteração do arquivo
  16. git reset HEAD nomedoarquivo - remove o conteudo do container
  17. git checkout -b teste - cria ramificação no grafo
  18. git log --graph --all - mostra o desenho do grafo com todos os ramos (se travar usar a letra Q)
  19. git checkout - volta ao branch identificado (volta todas as alterações do arquivo )
  20. git merge - une os branchs, deve estar no master e colocar o nome dou outro branch no caso teste
  21. git push - envia os arquivos local para o github
  22. HEAD - Ultimo commit

Observações: A cada commit o arquivo é atualizado, e não é criado nova cópia, ficando apenas o arquivo atual
